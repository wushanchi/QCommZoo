### 量子密钥分发（QKD）系统

#### BB84 协议（基础）
- [BB84 协议详解](https://blog.51cto.com/u_12868/14471310) - BB84 量子密钥分发协议原理与实现
- [BB84 光纤信道仿真](https://max.book118.com/html/2025/1225/8143022025010026.shtm) - 光纤信道误码率仿真与安全性评估
- [BB84 海底光缆](https://max.book118.com/html/2026/0204/8123064116010043.shtm) - 海底光缆通信中的性能评估

**BB84 协议原理：**
1. Alice 随机选择比特和基矢（直角或对角）制备单光子偏振态
2. Bob 随机选择测量基进行测量
3. 双方通过经典信道公开比较基矢，保留基匹配的部分作为原始密钥
4. 执行误码率检测，若超过阈值则判定存在窃听
5. 信息协调 + 隐私放大 → 安全密钥

**安全性基础：**
- 量子不可克隆定理
- 测量坍缩特性
- 任何窃听都会引入可检测的扰动

#### QKD 系统架构

**核心组件：**
- **量子态制备器** — 生成单光子或纠缠光子对
- **量子信道** — 光纤或自由空间传输
- **探测器** — 超导单光子探测器（SNSPD）或雪崩光电二极管
- **经典后处理** — 密钥协商、纠错、隐私放大

**技术挑战：**
- 光纤损耗（~0.2 dB/km）
- 探测器效率
- 偏振保持
- 信道噪声

### 量子纠缠与量子中继

#### 量子纠缠交换（Entanglement Swapping）
- [Qunnect + Cisco 实验](https://news.futunn.com/post/69002522/qunnect-and-cisco-demonstrate-first-metro-scale-high-speed-quantum) - 2026年2月实现首次城域规模量子纠缠交换

**实验成果：**
- 纠缠交换速率：**170万对/小时**（本地）
- 传输距离：**17.6 km NYC 光纤**
- 室温端点：可扩展部署
- 测试平台：Qunnect GothamQ（纽约）

**技术原理：**
1. 两对纠缠粒子（A-B 和 C-D）
2. 对中间粒子（B-C）进行贝尔态测量
3. 导致远端粒子（A-D）纠缠
4. 不需要直接传输纠缠粒子

#### 量子中继器架构
- [Quantum Repeater 分析](https://arxiv.org/html/2405.18252v1) - 量子中继链路的纠缠交换性能分析

**关键指标：**
- 记忆退相干
- 门操作精度
- 链路纠缠生成率

**量子中继功能：**
- **纠缠生成** — 在相邻节点间建立纠缠
- **纠缠交换** — 通过贝尔态测量延长纠缠距离
- **纠缠纯化** — 提高纠缠质量
- **量子存储** — 存储纠缠态直到链路建立

### 量子网络类型

#### 城域量子网络
- **应用场景**：城市内金融、政府通信
- **代表**：中国济南量子保密通信城域网

#### 洲际量子网络
- **挑战**：光纤损耗限制传输距离（~100km）
- **解决方案**：
  - 量子中继器
  - 卫星中继（墨子号系列）

#### 量子-经典混合网络
- **QKD + OTN**：量子密钥分发与光传送网融合
- **芯片化 QKD**：小型化集成，适配融合需求

### 量子通信协议

| 协议 | 年份 | 类型 | 特点 |
|------|------|------|------|
| BB84 | 1984 | prepare-measure | 首个实用化 QKD |
| E91 | 1991 | entanglement-based | 基于纠缠的 QKD |
| B92 | 1992 | prepare-measure | 简化版 BB84 |
| CV-QKD | 2000s | continuous-variable | 连续变量 QKD |
| MDI-QKD | 2012 | measurement-device-independent | 测量设备无关 |
