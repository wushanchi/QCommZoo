### QKD + 经典通信融合趋势

**行业共识**：QKD 与经典通信融合是未来趋势

- **国盾量子观点**：独立密钥分发网络（如干线或城域网）可能无需融合，但集成化是发展方向
- **技术方向**：
  - QKD + OTN（光传送网）融合
  - 板卡式 QKD（适配融合需求）
  - 芯片化 QKD（小型化集成）
